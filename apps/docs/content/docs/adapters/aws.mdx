---

Use `aws` for standard Amazon S3 buckets.

## Basic Setup

```ts
import { aws } from "vs3/adapters";

const adapter = aws({
  region: "us-east-1",
  credentials: {
    accessKeyId: process.env.AWS_ACCESS_KEY_ID!,
    secretAccessKey: process.env.AWS_SECRET_ACCESS_KEY!,
  },
});
```

## Options

`aws(options)` accepts AWS SDK `S3ClientConfig` options, except `forcePathStyle`.

Common options:

- `region: string`
- `credentials: { accessKeyId: string; secretAccessKey: string }`
- any additional `S3ClientConfig` option supported by `@aws-sdk/client-s3`

## Notes

- Use bucket region values like `us-east-1`, `eu-west-1`, `ap-southeast-2`.
- Keep credentials scoped to the minimum S3 permissions your app needs.
